# Peluditos 3D

Tienda estática de accesorios 3D para celular inspirados en mascotas. El sitio se publica gratuitamente con GitHub Pages y recibe pedidos mediante WhatsApp; no procesa ni almacena datos de tarjetas.

- Producción: https://peluditos3d.shop/
- Hosting: GitHub Pages
- Dominio y DNS: GoDaddy
- Fuente publicada: rama `main`, raíz del repositorio

## Desarrollo local

Sirve la carpeta con un servidor HTTP local y abre la URL indicada por el servidor. No pruebes el proyecto con `file://`.

La validación estática reproducible está en `tests/validate.mjs` y se ejecuta con Node.js:

```text
node tests/validate.mjs
```

## Datos comerciales pendientes

Antes de publicar políticas legales definitivas hacen falta textos aprobados por el propietario para privacidad, tratamiento de datos, cambios, devoluciones, envíos y términos. El sitio no inventa estas condiciones ni las presenta como vigentes.
